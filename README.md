# Python Sprint - Анализ данных

Проект для анализа данных с использованием Python, Pandas, и Jupyter Notebook.

## Установка

1. Клонируйте репозиторий
2. Создайте виртуальное окружение и установите зависимости:
```bash
python3 -m venv my_env
source my_env/bin/activate
pip install -r requirements.txt
```

## Настройка базы данных

1. Создайте файл `.env` на основе `.env.example`
2. Заполните параметры подключения к базе данных

## Запуск
```bash
source my_env/bin/activate
jupyter notebook
```

## Структура проекта

- `20.ipynb` - основной ноутбук с анализом
- `requirements.txt` - зависимости проекта
- `.env` - конфигурация базы данных (не коммитится)

## Технологии

- Python 3.11
- Pandas, NumPy
- Jupyter Notebook
